# 🔐 Random Password Generator

## 👤 Автор
**Имя и фамилия:** Несинова Вероника

## 📝 Описание
Приложение для генерации безопасных случайных паролей с графическим интерфейсом на Python/Tkinter. Позволяет настраивать длину пароля и типы используемых символов, сохраняет историю сгенерированных паролей в JSON.

## ✨ Функции
- Генерация паролей длиной от 4 до 32 символов
- Выбор типов символов:
  - Заглавные буквы (A-Z)
  - Строчные буквы (a-z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*)
- Копирование пароля в буфер обмена
- Сохранение истории паролей (последние 50)
- Экспорт истории в JSON
- Красивый современный интерфейс

## 🚀 Установка и запуск

### Требования
- Python 3.7 или выше
- Tkinter (входит в стандартную установку Python)

### Запуск
```bash
python password_generator.py
